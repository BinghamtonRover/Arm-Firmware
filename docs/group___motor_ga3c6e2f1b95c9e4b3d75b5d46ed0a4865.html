Checks if this motor has stalled.

Check this every STALL_CHECK_INTERVAL milliseconds. Use nextStallCheck to see when the next stall check has been scheduled for.

NOTE: Do not call fixPotentialStall from here, as that function calls this one to check for stalls, and will cause a stack overflow.
