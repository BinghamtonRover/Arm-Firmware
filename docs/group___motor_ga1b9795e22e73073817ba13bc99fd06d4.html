void StepperMotor::setup ( )

Initializes the motor.

If any part of the setup fails, the function will log to the serial monitor and block indefinitely. Issues in this function usually result from an issue with the chipSelectPin or the enablePin.

Many settings are configured here behind the scenes. For example, MOTOR_ACCELERATION and MOTOR_SPEED are both used here. In the future, explore what the different options actually do and how to optimize them.
